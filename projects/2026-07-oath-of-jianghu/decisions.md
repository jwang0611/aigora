# EditorInChief 决策日志 — 《三缄》

## [Checkpoint 1] Conception → Detailed Outline

**Round**: Conception round 1
**Question**: 故事内核 / 情感目标 / 结局方向 + Synthesizer 留下的两处分歧

**Options on the table**:
- A．换俘/换情报（宋金边境套路，Diverger 已否）
- B．坚壁清野——密令烧粮堵路，撤离窗口不足，义兄弟篡改文书争取时间，行为与"通敌"同构
- C．比武招亲变奏（情长过重，家国/义气沦为背景板）
- D．复仇路线（结构老套，遗憾落点不对）

**Decision**: 取 B。篇名《三缄》。

**Rationale**：B 是唯一一个不需要反派、单靠一个军事装置就能让三条线同时被抵押的方案
（原则1：最锋利的具体张力）。Literature Reviewer 指出的"忠奸对立"陷阱在 B 里被结构性
避开——郑帅的坚壁清野命令在纯军事逻辑上站得住，本篇的对抗不是忠奸对抗，是两种"家国
大义"定义的互相碾压，这比 A/D 的模仿性更强（原则4：具体、情境化的材料优于套路）。

- **故事内核**：义气、家国、情长常被写成互相成就的三位一体——为国尽忠即义气圆满，
  忠孝两全终成眷属。这篇写的是它们互相偿还不了：替一份尽了义，就还不了另一份的情；
  家国得存，代价却从史书里被抹去。真正的侠不是三者两全，是明知不可兼得，仍然选，
  然后终生不言——这也是篇名"三缄"的来源：三次沉默，三重代价。
- **情感目标**：不要"虽死犹荣"式的悲壮畅快，要一种清醒的、无处偿还的亏欠感。读者合上
  小说时想起"三缄"二字，心口发紧，而不是热血上涌。
- **结局方向**：不昭雪、不表白、不团圆。顾长风活下来，背着"通敌"的官方定论终生不辩白——
  这个定论此刻甚至部分为真（他后来确实接手扩大了违规操作），他不辩白不是因为怕，
  是因为辩白必须牵出郑帅坚壁清野的原始命令，而战后这道命令已经被写成"关隘保卫战的
  胜利决策"——揭穿它不会还沈砚和苏晚公道，只会让更多人受牵连。这是他最后一次、
  也是最沉重的一次"三缄"。

**分歧裁决**：
1. **苏晚死因**：取"坍塌"，不取"北虏斥候误伤"。理由：坍塌由沈砚点燃调虎离山之火
   蔓延到药庐街区直接导致，因果链完全自洽在"这一晚三人各自的选择"内部，不需要引入
   外部的第四方（北虏斥候）来收尾——外部力量收场会稀释"没有反派，只有装置"的核心命题
   （Challenger 的理由更贴合故事内核，采纳）。沈砚之死改为：他为拖住郑帅的搜查队，
   独自炸断身后的粮道栈桥，被随后波及的坍塌一并压在桥下——这样苏晚与沈砚死于
   **同一处装置的两次坍塌**（药庐街区火势蔓延 / 粮道栈桥自毁），呼应更紧，也仍然避免
   郑帅或北虏充当直接处刑者。
2. **结局是否确认郑帅"看穿真相"**：取"不确认"，只给一个极淡、可以两种方式解读的动作
   （判词上"流放"二字改判前的一瞬停笔），不给台词、不给眼神交流的明写。理由：本篇的
   基调是"沉默"，任何一处明写的"心照不宣"都会把沉默变成温情脉脉的默契，削弱"家国吃掉
   了证词"这一命题的冷度（Challenger 原判据成立，采纳）。

**Overruled**：Yes And 提议在结局给郑帅一句暗示性台词（"记录以外的话，本帅不听"）——
拒绝。任何台词都会把留白坐实，变成一次隐蔽的表白，违反上面裁决2的理由。改为纯动作
（停笔一瞬、改判词时笔尖的一个停顿），不给语言。

## [Checkpoint 2] Detailed Outline → Drafting

**Round**: Outline round 1
**Question**: 结构确认 / POV / 必须存在的时刻

**Decision**:
- **结构**：五节——《暮年》《义结》《粮册》《青峡雪》《三缄》。首尾为"暮年"框架
  （约500 / 约900–1,100字），中段三节线性直叙，不加更多倒叙层。总体量约
  8,000–9,500字。
- **POV**：第三人称有限视角，锚定顾长风，过去时；克制、干净，不直接命名情绪，
  全篇不使用感叹号，不写"我爱你"及其任何变体。
- **必须存在的时刻**（六项，见 outline.md「Key Moments」）：少年之约原话；苏晚拒绝
  优先转移有身份伤员的旧例场面；顾长风"三天也不够"的主动转折句；沈砚被打断、永不
  给下文的未尽之言；郑瀚落笔前的一瞬停顿（不给台词）；铜哨与白芷首尾呼应。
- **PlotArchitect 移除测试通过项**：沈砚暗恋苏晚（不可删，义气与情长的钉子）；郑瀚
  私下宽宥的唯一动作（不可删，体制内部也有不能言说的仁慈）。

**Rationale**：五节框架结构与《慢房间》先例一致（首尾环抱、中段线性），适合短篇体量
下容纳三线交织而不显仓促。POV 锁定顾长风而非多视角，是为了让"苏晚不知情"和"沈砚
未尽之言"始终只能通过缺席被感知，不被作者用视角特权泄露——这是 Challenger 在
conception 阶段定的红线（不给煽情自白）的结构化落实。

**Overruled**：ScenePainter 倾向在第4节坍塌场面加入更多环境描写渲染惨烈（火光、
哭喊的铺陈段落）——部分拒绝。允许保留必要的空间方位交代（读者需要知道两处坍塌
分别在哪、顾长风为何"都赶不上"），拒绝渲染式的苦难奇观描写，保持 story-bible
"不写多余场面"的语体戒律。

## [Checkpoint 3] Critique Round 1 → Revision

**Round**: Critique round 1（盲评中位数 43/60，REVISE；Originality/De-AI 中位数 6，
触发单项下限）
**Question**：修订清单取舍

**Decision**：执行一轮定向修订，修订后重跑一轮全新盲评（不沿用本轮评分）以验证是否
过线。

**Accepted — Must Fix（四项，取三份评审并集，全部无条件接受）**：
1. **重做第4节坍塌成因的因果机制**：不再用"风向偶然+此前未提过的第二处柴堆"制造
   苏晚之死，改为：第3节顾、沈二人选定纵火点时，明确权衡并接受"栈桥柴垛顺风可能
   烧向城墙根干柴堆、进而威胁回春堂街区"这一已知风险，并为此专门安排一名更夫值守、
   一旦风向不对即示警；第4节行动之夜，这名更夫恰恰是被抽调去加强疑兵搜查方向的
   人手之一——他们自己安排的调虎离山"太成功"，反而抽空了自己留的安全阀。灾难由此
   变成一次被清楚认知、仍然接受、最终反噬的赌注，而不是纯粹的天气意外。
2. **压缩/改写第5节"第三次沉默"的编号式总结**：删去"第一次是……第二次是……第三次
   是……"的清点句式，让"三缄"二字仍在文本中落地一次，但不做主题解说，让此前已经
   完成的意象（哨子、供词、终身不辩白）自己承担分量。
3. **给苏晚补一层独立于两位男主的能动性**：第2节增补一处她私下的疲惫/自我怀疑的
   瞬间（被顾长风无意撞见），交代她坚持这条旧例的私人缘由，而不只是一条服务后续
   剧情的规矩。
4. **为郑瀚的从轻发落补上性格铺垫**：第2节下令时增补一句他"赏罚从不看情面，也从
   不多罚一分"的军中风评；第5节的改判，落笔前的停顿仍不给台词、不确认是否看穿
   真相，但读者现在有一条可依据的性格线索——他可能只是按顾长风供词字面上"未经授权
   擅自下令疏散"定罪，而非按"通敌"顶格量刑，这在他一贯"只认凭据、不诛心"的作风里
   说得通。

**Accepted — Should Fix（全部采纳）**：
- 第2节补一处铜哨信号的少年起源（一句话量级）。
- 全篇"不是A，是B"句式从至少5处削减到2处以内，其余改写。
- 削减/改写解释性明喻（拂灰动作、苏晚死亡姿态两处优先处理）。
- "他后来想/常想"回溯装置从5处以上削减到2处以内，其余改为即时场景呈现。
- 第4节苏晚转移伤员的表述补一句澄清，避免与第2节旧例产生误读式矛盾。
- 第2节人物/军令/量刑的密集信息适当拆开节奏，不要求大改，允许在段落间补一到两处
  呼吸点。
- 第3节"苏晚知不知道"一段，为顾长风补一处更早、更克制的内心闪现，减少"情长"线
  完全依赖事后追认的问题。

**Overruled（本轮不采纳）**：
- R3 Optional"铜哨缺口来历"——拒绝。缺口作为纯氛围留白，补来历会把一个克制的物证
  变成一次多余的注解，违反"不写多余场面"戒律。
- R3 Optional"护卫队正军衔重复"——拒绝。非必要重复，不产生新信息。
- R2 Should Fix"重新考虑沈砚未尽之言的时机"——部分拒绝。桥面自爆前的最后一句话，
  在"这是他唯一剩下的时刻"这层意义上是站得住的固有惯例，不属于可无成本替换的套路
  项；本轮不改时机本身，只随因果机制重做（见 Must Fix 1）一并检查前后文是否顺畅。

## [Checkpoint 4] Critique Round 2 → Revision（含同批次追加修订）

**Round**: Critique round 2（盲评中位数 42/60，REVISE；Originality/De-AI 中位数 6，
仍触发单项下限）
**Question**：修订清单取舍

**Decision**：本轮盲评发现 Checkpoint 3 的修订仍不够彻底——尤其是火灾成因，R2/R3
仍判定为"运气+无关配角的被动服从"，未达到"主角自己的选择造成代价"的标准。执行第二轮
定向修订，其中一项改动（火灾成因重做）幅度较大，在三份盲评已经分发之后才最终定稿，
故本轮盲评三份报告读到的是追加修订前的快照——追加修订的内容已记录于
continuity-ledger.md「Round 3 Revision Deltas」，此处只记裁决。

**Accepted — Must Fix（五项，全部无条件接受）**：
1. **火灾成因二次重做**：不再是"郑瀚加派的无名巡查队强征张老栓"，改为顾长风本人的
   临场决断——搜查队被调去粮道后，郑瀚另加派一队盘查回春堂这条街，顾长风为保住
   疑兵不露馅、又抽不出别的人手，亲手把张老栓调去应付这道盘查，两害相权，自己放下
   了柴堆这层风险。灾难由此完全系于主角自己的选择，而非配角的被动服从或纯粹运气——
   直接回应 R2 两轮评审共同的核心质疑。
2. **军令期限由"三日之内"改为"十日之内"**，修正与实际侦查/对质/行动跨度（约十日）
   不符的时间线矛盾；沈砚的窗口测算相应改为"离限期只剩三日"的相对表述，原有数字
   （一日半缺口、四日半总需求）不变。
3. **供词笔迹证据链补漏洞**：第3节新增顾长风提出方案后主动接手此后所有批文誊写、
   用印；第5节军法官翻出的正是他接手后亲笔的那几批，沈砚更早的亲笔账册始终没有
   被翻出——解决"顾长风声称从头到尾一人所为，但物证上仍有沈砚笔迹"的矛盾。
4. **修正第4节视角越界**：药庐内部苏晚指挥转运一段，改写为顾长风事后从伤员、药童
   处拼凑而知，不再是他未曾进入房间却直接掌握的全知细节。
5. **补 Gu-Su Wan 情感线的更早铺垫**：第2节增补——沈砚说"这只手比他师父的手稳"后，
   顾长风自己也在廊下看了那双手很久，此事未告诉任何人，包括沈砚——与沈砚的暗恋
   对称，解决"顾长风的情感只能靠拂灰这一个事后动作追认"的问题。

**Accepted — Should Fix**：
- 削减"他没想到/后来常想"先知式懊悔句（5处→2处）。
- 削减"不是A，是B"句式（narrator声部5处→2处，苏晚"不是官大的先走"作为人物固定
  台词保留不计）。
- 风向伏笔明写"北风"，与第3节"若那晚起了北风"对应。
- "没提一句沈砚"改为"没有主动提沈砚一个字"，避免与后文直接回答矛盾。

**Overruled（本轮不采纳）**：
- R2 Should Fix"系统性削减破折号警句结构（15+处）"——部分拒绝，仅做局部处理。理由：
  全篇重写破折号句式的收益边际递减，且部分被引用的例句（"不是没人心软，是这笔账"
  "这个动作他活了三十多年，从没在她面前做过"）本身被同一份评审判定为"最好的句子"，
  不宜为了统一削减而误伤——本轮只处理确属懊悔式/说教式的破折号句，其余保留待人工
  在正式出版前视情况精修。
- R2 Should Fix"让沈砚说完那句未尽之言，取消打断结构"——拒绝，理由同 Checkpoint 3：
  这是全篇"三缄"母题最直接的一次具身化，取消会削弱篇名与结构的呼应，risk 大于收益。
- R3 Should Fix"郑瀚终生不再与顾长风说话"的代价应更"可见"——拒绝。这一沉默本身就是
  故事对"体制内部不能言说的仁慈"这一命题的具身化处理，让它"可见"会违反第4节的
  语体戒律（不给台词、不给场面）。

## [Checkpoint 6] Critique Round 4（最终核验）→ 升级人类作者

**Round**: Critique round 4（对应规则允许的第3轮修订之后的核验；中位数 45/60，REVISE；
六维度全部≥7，无单项下限问题）
**Question**：是否继续修订，还是按规则升级

**触发条件确认**：这是第3轮定向修订之后的核验评审。按 `quality-rubric.md`「短篇门槛」
与 `editor-in-chief.md`「必须升级情形1」：质量门在3轮修订后仍未过线，应停止修订，
向人类作者呈报评分历程、诊断与选项，而不是自行开启第4轮全新盲评。

**决策**：不再开启新一轮三路盲评。但在升级前，先对 R2（本轮唯一给出新 Must Fix 的
评审）遗留的三项 Must Fix 做最小必要修订，理由：这三项都是具体、可核实、低风险的
精确性修订（消除一处可被读作矛盾的歧义表述；把一句站不住的"这不是……却也不是谎言"
改写成对"顾长风确实撒了一个谎"的正面承认；给风险决策补一处他本人已经看见风向转变
仍下决断的细节），不属于"为冲分而磨平声音"的注水式修订，符合主编"可以在升级前
执行低风险精确性修正，但不得为逼近48分而反复重开盲评"的自我约束。此外一并修复了
R1本轮独立发现的一处连贯性硬伤（第4节"两次爆炸"）。

**Accepted（三项，均为最小必要修订，不重开盲评）**：
1. 第5节"更早几日沈砚亲笔核算的那一本,始终没有被人从底下翻出来"，改写为明确追溯到
   郑瀚第4日亲手点破的那一批——顾长风连夜连同该批一并重誊、旧本尽数烧毁，"这世上
   再没有一页留着"，消除歧义。
2. 第5节"这不是全部的实情,却也不是谎言"改写为正面承认："整篇供词，他只在这一句上
   说了谎，说得面不改色"——不再用叙述者的口吻替他的道德立场做粉饰。
3. 第4节顾长风调走张老栓前，补一处他已经看见风向偏北、雪势转密，自己压下不安仍
   下决断的细节，强化选择属性。
4.（R1本轮发现，非R2项）第4节"闷响……是炸药的动静"改写为搜查队围人的打斗声响，
   消除与后文栈桥真正引爆之间的"两次爆炸"混淆；第3节天数换算精简，减少非必要心算。

**Overruled / 留待人类裁决而非本轮处理**：
- R2 Should Fix 全部四项（补沈砚风险的更早铺垫、补苏晚独立于两位男主的性格切面、
  给张老栓一个非工具性的细节、把郑瀚法理权衡段改写得更戏剧化而非分析化）——不在
  本轮处理。理由：这些是质量打磨项，不是阻断项（六维度已全部≥7），继续处理需要
  再次进入"改动→需要重新盲评验证"的循环，而规则规定的3轮修订上限已经用尽。

**呈报给人类作者（brief 未预留此项决策，但主编判断"是否以当前版本定稿"触及创作
风格取舍，主动呈报而非自行拍板）**：

- **评分历程**：43 → 42 → 45 → 45（4轮中位数合计，满分60，通过线48）。
- **诊断**：三位评审员的判断持续分裂。R1（读者视角）与 R3（结构/知识一致性）两条
