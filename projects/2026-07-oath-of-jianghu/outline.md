# Outline — 《三缄》

**Structure**: 五节；首尾为"暮年"框架（约500字 / 约900字），中段三节线性直叙，
不再夹更多倒叙层。总体量约 8,000–9,500 字。
**Locked at**: Outline checkpoint（见 decisions.md Checkpoint 2）

## Chapter/Scene List

### 1. 暮年（框架·开篇，约500–600字）
- **Situation → change**：老年顾长风，脸上刺字未除，独居在一处荒坡边，照看一片野生
  白芷，贴身带着一枚铜哨。一个路过的年轻游骑向他打听"当年青峡关一战"，他只说了官方
  记得的那部分。
- **Reader learns / feels**：他是个背着通敌污名活下来的老人；有桩往事被他刻意留白。
  不解释白芷和铜哨是什么，只让它们"在场"。
- **Ledger-critical facts established**：顾长风脸上有刺字（通敌罪的标记）；青峡关一战
  官方记载为"守关胜利"；他随身有铜哨、门前种白芷。

### 2. 义结（约1,700–1,900字）
- **Situation → change**：倒叙至三十余年前——不，改为十余年前（顾、沈已成年任职，
  结拜是更早的少年往事，此节以简笔追叙带过，重心落在"当下"：郑瀚下达坚壁清野密令，
  顾长风奉命协防撤离，沈砚以主簿身份核算发现撤离窗口不足。同节内插叙结拜时"谁都不许
  先惊动苏晚的心"的少年之约，与苏晚"按伤重不按身份"的旧例（她当众拒绝优先转移一位
  伤员家属）。
- **Reader learns / feels**：三条线的地基在此节全部埋下——军令的铁律（规则1）、少年
  之约（规则3）、药庐旧例（规则2）。读者应感到"这三件事本该互不相干，却共享同一批人"。
- **Ledger-critical facts established**：坚壁清野密令内容与量刑（私改文书=通敌，斩）；
  少年之约原话；药庐旧例与苏晚拒绝优先转移某伤员家属的具体场面。

### 3. 粮册（约1,700–1,900字）
- **Situation → change**：郑瀚察觉撤离批文有异，命顾长风彻查"内鬼"。顾长风循线查到
  篡改者是沈砚，深夜对质。沈砚坦白：药庐还困着近百挪不动的重伤员和苏晚，窗口期内根本
  运不完，他篡改文书是为了多争三天。顾长风没有立刻决定告发或包庇，而是自己重新核算
  了一遍撤离表——发现即便三天也不够，除非再制造一次调虎离山，把郑瀚的搜查队引离粮道。
- **Reader learns / feels**：读者此时应意识到，顾长风一旦参与重新核算并提出调虎离山的
  方案，他自己已经从"查案人"变成"同案人"——这不是被沈砚说服的软弱时刻，是他主动的
  技术判断（"三天也不够"）逼出的选择。
- **Ledger-critical facts established**：撤离窗口的具体数字（人数/天数）；顾长风提出
  调虎离山方案；顾长风此刻起的行为已构成他后来"顶罪"时供词的事实基础（非谎言伏笔）。

### 4. 青峡雪（约2,000–2,300字，全篇高潮）
- **Situation → change**：行动之夜。沈砚点火制造调虎离山的疑兵，引开郑瀚的搜查队，
  途中被逼至粮道栈桥，炸桥断后自陷桥下——他对顾长风说了一句没说完的话（被爆破声打断）。
  火势蔓延到药庐街区，苏晚坚持按旧例先转移挪不动的重伤员，药庐坍塌，她死于其中。
  顾长风在两处之间往返，都只赶上"之后"。
- **Reader learns / feels**：不给"来得及"的侥幸，两处都只写"到场时已经发生"。沈砚
  未说完的话和苏晚的死同时降临，情感重量在于"没有一处来得及"，不靠煽情句子，靠时间
  差的事实排布。
- **Ledger-critical facts established**：沈砚死于粮道栈桥坍塌；苏晚死于药庐坍塌；
  沈砚未说完的话的确切断点（"其实我一直……"被爆破声打断，全篇不给下文）；铜哨遗留
  原地，被顾长风拾起。

### 5. 三缄（框架·结尾，约900–1,100字）
- **Situation → change**：天亮后军法官审理，顾长风主动交代篡改文书、组织调虎离山
  皆出自己手（此刻已是事实）。郑瀚提笔拟斩，落笔前一瞬停顿，改判"流放，刺字发配"，
  不给解释。回到开篇的"暮年"时空：顾长风终生未再向任何人提起那晚的全部经过，青峡关
  一战被写入战报为"顺利完成坚壁清野，一举挫敌"。他把白芷移种在药庐旧址，铜哨终生
  不再吹响。末段"三缄"二字随叙述自然落地。
- **Reader learns / feels**：亏欠感而非悲壮感——他活着，赢了这一仗，却输掉了能说出口
  的一切。
- **Ledger-critical facts established**：判词由斩改流放的确切动作（停笔一瞬，无台词）；
  官方战报措辞；顾长风终生不辩白的选择及其理由（辩白会牵出郑瀚原始密令，让更多人受
  牵连）。

## Key Moments（必须存在的时刻）
1. 少年之约原话——"谁都不许先惊动她的心"（第2节）。
2. 苏晚当众依旧例拒绝优先转移某位有身份的伤员（第2节，为第4节铺垫）。
3. 顾长风重新核算撤离表后说出"三天也不够"——他主动变成同案人的转折句（第3节）。
4. 沈砚被爆破声打断、没说完的那句话（第4节，全篇不给下文）。
5. 郑瀚落笔前的一瞬停顿，改斩为流放（第5节，不给台词）。
6. 铜哨与白芷在开篇与结尾各自呼应一次（第1节 / 第5节）。

## Ending
末段：顾长风蹲在药庐旧址新移种的白芷丛边，年轻游骑已经走远。他从怀里摸出那枚磨得
发亮的铜哨，没有吹，只是握了一握，重新放回怀里。叙述句点出：他这辈子学会的最后一件
事，是有些账，认了就是还了——不必让人知道，也无人可说。三缄其口，是他此生最后，
也最沉的一次义气。
