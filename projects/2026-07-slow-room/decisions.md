# EditorInChief 决策日志 — 《慢房间》

## [Checkpoint 1] Conception → Detailed Outline

**Round**: Conception round 1
**Question**: 故事内核 / 情感目标 / 结局方向

**Options on the table**:
- A 记忆修复（真实 vs 仁慈）
- B 声音银行（语言之死，更安全更雅）
- C 慢房间（告别的不对称：门外一小时，门内十年）
- D 告别技术泛滥的讽刺喜剧

**Decision**: 取 C。题名《慢房间》。

**Rationale**: C 的张力最具体且无处可逃——不需要任何反派，规则本身就是刀（原则1：最锋利的具体张力）。
Challenger 说 B 更安全，正因如此不取（原则2：情感风险更高且讨论已证明能落地——特德·姜视角给出的三条
推论链保证了它不是空转的煽情）。D 的讽刺赢得回合，输掉故事。

- **故事内核**：时间无法共享；爱是把不能共享的时间仍然朝向对方。技术没有解决亏欠，只是给了亏欠一个
  足够大的房间。
- **情感目标**：第4节，被先行哀悼的荒凉；第5节，方向相反的第二次崩塌。终感为"grief with resentment,
  love without settlement"——读者合上时既酸又暖，但账没有平。
- **结局方向**：**不完全和解**。独白记录证明十年朝向她，但迟到本身不可改写（世界规则2 的主题化）。
  最后一行不给宽恕，给的是"收到"。

**Overruled**: Yes And 提议让父亲与记忆中亡妻的共同生活占更大比重（讨论中获多数附议）——降为一条
支线细节。理由：这篇的轴是父女的账，妻子线加重会把小说推向"团圆补偿"，稀释不完全和解的结局。

## [Checkpoint 2] Detailed Outline → Drafting

**Round**: Outline round 1
**Question**: 结构确认 / POV / 必须存在的时刻

**Decision**:
- **结构**：五节——《同意书》《迟到》《一小时》《回来的人》《独白记录》。第2节整节倒叙，其余线性。
- **POV**：女儿第一人称，过去时。审计式语体：精确、数字化、情绪永不命名。全篇不使用感叹号。
- **必须存在的时刻**：
  1. 签字页上"独白记录：导出/不导出"的勾选框（第1节埋，第5节起爆）；
  2. 门外一小时中她换算时间比率的心算（1分钟≈61天）；
  3. 父亲醒来后的第一句话必须是错位的日常（十年时差的具体化），不许是任何抒情句；
  4. 独白记录中"同一趟车重开了很多遍、每一遍都迟到"的段落（规则2：事实不可改写）;
  5. 没送出的棉拖鞋。
- **体量**：五节各约 1,800–2,400 字，总 ~10,500 字。

**Rationale**: 五幕结构在 PlotArchitect 方案上无修改必要；第3节的"薄"用双层结构解决（实时等待 ×
医生解说的设定推论交错），并把心算移入此节作骨。POV 与语体是煽情风险的主要防线（原则4：具体、克制
的声音优于抒情腔）。

**Overruled**: ScenePainter 倾向在第4节加入病房外自然景物的抒情段——拒绝。理由：违反语体决定，
第4节的力量必须全部来自对话的错位，不许借景。

## [Checkpoint 3] Critique → Revision

**Round**: Critique round 1（盲评中位数 51/60，PASS；R2 单独 46）
**Question**: 修订清单取舍

**Decision**: 执行一轮定向修订，之后重跑一轮完整盲评以验证（自加严：门槛只要求
Must Fix 修复，但 R2 单独未过线，重评更稳妥）。

**Accepted — Must Fix 全部 8 项**（见 scores.md）。关键方案：
- 导出件体量：走"话少的人档案也薄"方向——3,650 个文件、37 万字、压缩包一兆多；
  秦大夫台词改为"有人说个不停"；第5节点明他在里面也省字。媒介统一为语音转写，
  删除"他自己写着"类字样。
- 勾选框改为归患者处置（导出给家属/销毁），与父亲勾选动作一致。
- 规则边界：第1节秦大夫补一句"重演真实记忆，结局改不了；重组和想象随便"；
  第5节补一句呼应（想象里随便开，一进那晚的记忆方向盘就不听）。
- 时率方向、车程数学（4h59/62km/h）、3,650天/2,555段 全部改齐。

**Accepted — Should Fix（R2 ①②④⑤⑥⑦⑧⑨⑩ + R1⑤ + R3①④，部分为修改后采纳）**：
- 删"这回没迟"（信任读者，"爸到了"独立承担闭合）
- 末件嘱咐删"买双厚拖鞋"直点，留"你那屋子地凉"（镜像由玄关拖鞋完成）
- 监狱试点（撞《黑镜》）换为"健康人想买延寿被卡死"（同时加固规则4）
- "不是X是Y"句式 4→2；"平得像X"明喻 3→1；审计术语自我标注 4→2
- 第2节缺席清单破对称（加录取通知书贴驾驶室细节）；删章尾"下集预告"句
- 母亲加一个不驯服细节（砸车钥匙又胶带缠好）
- "她让我先走"改为"走的时候我把门带上了"
- 第3节删"我忽然明白我在生什么气"自诊断句
- "年均1.7次×34年"统计口径修正；"信阳2,200次"与"重开两千多遍"口径钉死
- "十九天前确诊"与"72小时"时点补一句（三天前改的数字）
- 第4节"送完了"后补一句钉死"送≠消失"（送是送，过是过）

**Overruled（拒绝项，记录在案）**：
- R2 Optional"故意留毛边/降低伏笔回收率"——拒绝。人为弄脏是另一种表演；
  本篇的"过度整洁"是叙述者人设（审计员）的自然结果，保留。
- R3 Optional"拖鞋购买时点"——非缺陷：拖鞋为病房日常所购，与慢房间无关，不改。
- R2 Optional"父亲金句密度补丁"——部分处理（删2句），不做系统性稀释；
  门内十年的人语言密度升高有设定支撑（他练了十年独白）。
