# Outline — 《慢房间》

**Structure**: 五节；第2节整节倒叙，其余线性。总 ~10,500 字。
**Locked at**: Checkpoint 2（见 decisions.md）

## Chapter/Scene List

### 1. 同意书（~2,000字）
- **Situation → change**: 临终关怀中心，秦大夫讲解慢房间规则与风险，父亲决定进入；
  女儿作为唯一家属签字见证。她反对，但父亲用他一贯的方式（不容商量的具体安排）压过。
- **Reader learns / feels**: 全部世界规则（1/2/3/4/5 自然嵌入讲解与问答）；父女间有旧账，
  但账目内容按下不表。
- **Ledger-critical facts**: 签字时间 14:03；启动定在次日 09:00；父亲预期生存 ≤72小时；
  独白记录勾选"导出"（女儿没细看勾选框——她以为他不会勾）；7% 走失率被父亲问了一句
  "走失了算谁的"；棉拖鞋尚未出现。

### 2. 迟到（~2,200字，倒叙）
- **Situation → change**: 母亲的死（14年前）：尿毒症末期最后一夜，父亲在沪昆高速，
  差四十七分钟。此前三十年货运生涯的"不在场"清单；此后14年父女的疏远账目
  （每年见面次数、转账记录、通话时长——用审计语言列）。
- **Reader learns / feels**: 怨恨的来历是具体的、可核对的；父亲从不辩解也从不道歉。
- **Ledger-critical facts**: 母亲死时女儿20岁；父亲当时距医院312公里；此后14年父女
  年均见面1.7次；父亲退休后学会用微信但只发语音从不打字。

### 3. 一小时（~2,200字，双层结构）
- **Situation → change**: 09:00–10:00 实时等待 × 她的心算与想象交错。她算：1分钟=61天；
  她去买了咖啡错过的9分钟=他的一年半。中段插入秦大夫关于"门内生活"的解释（重演、重组、
  部分人建造惯例度日）。她带来的袋子里有43码棉拖鞋——中心规定门内启动后不得再进病房，
  没送成。
- **Reader learns / feels**: 被抛下的感觉第一次具象化——他正以她的六十倍速离她而去；
  她开始后悔签字。
- **Ledger-critical facts**: 启动 09:00 整，时长 60 分钟；她 09:22–09:31 离开走廊买咖啡；
  棉拖鞋 43 码、蓝格子、118 元。

### 4. 回来的人（~2,200字）
- **Situation → change**: 10:00 出来，父亲醒来，存活以小时计。十年时差的错位对话：
  第一句是错位的日常（他问的时间对不上）；他把门内生活当实事讲（和她妈过日子、跑车、
  "你妈让我告诉你灶上的火别虚着"）；他平静到陌生。压垮她的一句：他说他在里面
  "把大家都送完了"——包括她。她意识到自己被先行哀悼。当夜父亲死亡。
- **Reader learns / feels**: 荒凉——告别的不对称完整落地。读者此刻应站在女儿一边：
  他又一次先走了。
- **Ledger-critical facts**: 父亲醒后存活约9小时，死亡时间 19:41；他门内主观年龄约72；
  对话中提及门内的"家"设在1998年的记忆地形里；他没有道歉。

### 5. 独白记录（~2,000字）
- **Situation → change**: 葬礼后第9天，中心发来独白记录导出件：3,652 个文本文件，
  共 37 万字，按门内日期命名。她以审计员的方式读：抽样 → 通读。
  - 头两年多：同一趟车。他把那晚沪昆高速重开了很多遍（她数了：两千多遍），每一遍
    都差四十七分钟——记忆里的事实不可改写。某一天的文件只有一行，标志他停下了。
  - 中段：惯例。和记忆里的母亲过日子；跑没有货的车。
  - 后七年：每个文件末尾多出一段——对着她说的话。每天一段，讲他这一天，问她吃了没。
    3,652 天里有 2,557 段。他知道她听不见；他照说。
- **Reader learns / feels**: 反向崩塌。十年不是逃离她，是每天回来一次。
- **Ledger-critical facts**: 最后一个文件（门内最后一天）内容：他说今天不跑车了，
  在门口坐一会儿，等到点了就出去；末句对她说——与棉拖鞋呼应的一句日常（不许抒情）。
- **Ending**: 她的"签收"：不写宽恕。她做了一件审计员式的小事（把 3,652 个文件按年份
  归档备份三处），最后一行落在棉拖鞋的处置上——具体、日常、方向朝他。

像家里有人要回来换鞋。末句是审计式的一行，不解释。
