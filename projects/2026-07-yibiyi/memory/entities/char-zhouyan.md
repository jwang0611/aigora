# Entity — char-周晏

**Type**: character
**One-line**: 30出头，返乡迁自家祖坟的本地人，兼工程队本地联络；对"留不住"看得最平。
**Voice notes**: 干活的话，实在，带点黑色幽默；不感伤；说"迁""挪""了断"等动词，不说"记住"。

## Through vol 1（起始状态）
- 早年外出打工，为赶在蓄水前迁走三代祖坟而回。与江照是搭档式同行——他挖坟，她扫村。
- 是江照的镜子：同样面对失去，他选接受、了断；她选记录、留存。
- 关系：与钟阿婆是旧识晚辈；与江照有克制的同行情谊，不越界。

## End of vol 1（D43）
- **Location**: 南屿下游，帮工程队清最后无主坟。
- **Knows**: 已迁完自家三代祖坟（"了了"）。
- **Open wants**: 干净了断；不留念想。（镜子功能：以"接受"衬江照"留存"。）
