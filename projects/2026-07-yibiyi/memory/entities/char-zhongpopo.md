# Entity — char-钟阿婆

**Type**: character
**One-line**: 钟秀英，78，南屿最后的留守者、自封的守碑人；手抄《浔口名册》记谷中所有死者，含1970s无坟溺者。
**Voice notes**: 方言味短句；爱用称量词（"一笔""一画""压秤""值不值"）；不解释，只陈述；认号牌那刻话最少。

## Through vol 1（起始状态）
- 独居谷中老屋，拒绝被扫描。弟弟**钟满**，13岁，1970s老浔口放水那夜溺死，无坟——名册里一条只有名字、
  没有葬处的空条。
- 守着淹没线下的碑与名册；工程队视她的碑为"障碍物"。
- 关系：起初把江照当"扫村子的人"防着；渐生亲近；尚未认出号牌/江照身世（v1末仍未全知）。
- Open wants：让谷里每个死者"有名有落"；怕自己一走、名册一散，无坟者第二次没顶。

## End of vol 1（D43）
- **Location**: 南屿谷中老屋/碑前，不肯离谷。
- **Physical state**: 78，握笔的手。
- **Knows**: 谷中所有死者的名与落；老浔口1976溺者名册；**不知**江照身世、不知号牌。曾轻言空条里"押着我自个儿的人"（弟钟满，未对江照明说关系）。
- **Relationships**: 与江照——容她看名册、比日子。
- **Open wants**: 名册有人签收、接续；无坟者有名有落。
