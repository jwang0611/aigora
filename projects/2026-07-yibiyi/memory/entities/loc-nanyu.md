# Entity — loc-南屿 / 老浔口

**Type**: location
**One-line**: 西南浔江河谷的南屿村；2029年将被南岭引水枢纽二次淹没；下游老浔口已于1970s被上一座坝淹过一次。

## Through vol 1
- **南屿村**：谷中村落，青石老屋、晒场、祠堂、后山坟地。100天倒计时（v1约剩40天）。红漆**淹没线**
  刷满墙、树、碑；线下的坟须迁。多数人已迁走，剩钟阿婆等极少数留守。
- **老浔口**：下游老镇，1970s小坝放水时淹没，当年溺死数人、多无坟。江守（江照祖父）是那夜的摆渡人。
  如今在更深的水下——南岭蓄水后将被"淹第二次"。
- **数字南屿**：外包的 1:1 数字孪生副本（激光扫描+摄影测量），几何/材质/光照俱全，点云空心。
- 现实纹理：迁坟、淹没线红漆、库区移民、二次淹没、退水期露出的旧镇轮廓。

## End of vol 1（D43）
- 倒计时约40天；南岭开始下闸蓄水，水从下游一寸寸顶上。外业扫描基本收尾（钟阿婆屋/人未扫）。
- 未解：祠堂后进正梁北端补不上的黑（朝坟地）。
- 老浔口：蓄水后将在水下再深约40米（二次淹没）。
