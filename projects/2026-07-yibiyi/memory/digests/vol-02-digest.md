# Volume Digest — 《一比一》 vol 2: 没顶

<!-- HARD CAP 2,500字。全书收官卷。 -->

## What this volume did for the book（一句话）
把"留不住"坐实，让机器留不下的那一样东西——一个无坟者的名字——由一个人接过来、带过水、签了收。

## Plot summary（按章簇）
- **ch07 退水（D38）**：南岭分级试蓄，下游老浔口旧河床短暂露出（断墙、义渡亭石脚、渡口木桩七根，桩顶高程约251）。江照第一次实地站上"江家的谷"，扫断墙残碑（含何广仁碑），未取号牌比对。村空至仅剩钟阿婆一户。
- **ch08 交底（D33）**：江照对钟阿婆交底、示号牌"浔渡贰号"、认江守之孙。"江"姓空条＝**江有田**（下街种田户，经县旧档核证非江照亲族；名册仅此一个江姓）——red herring 收口。江家那夜整户在船、无人溺，少的是钟满等钟家人。钟阿婆不控诉不宽恕、早有觉察（"对上了"），把号牌搁在江有田那条、不挨钟满行。号牌是否即那趟船，留白。
- **ch09 数字南屿（D25，水位约282）**：陆携市里验收数字南屿，戴眼镜"回到"空心副本，官员赞"一比一/留住了"。祠堂正梁北端那片黑被补面抹平（覆盖率99.7%），江照默许未阻止（留憾），信念起裂。章末：她把北端未补的原始点云单存一份、文件名留空。
- **ch10 最后一渡（D10，水位约285）**：钟阿婆在渐涨的水边做最后一次"送"，念1976无坟14名含钟满，江照持灯替念。钟阿婆把《浔口名册》交江照"签收"（非赦非罪，口授记册规矩：无凭处留空、衣冠冢须验看）。她**未殉谷**——晚上八点多搭周晏带车斗的车离谷（樟木箱+被+包袱，留下黑铁锅）。
- **ch11 没顶（D0，合闸后水位302→308→326）**：南屿自下而上没顶（晒场286先、祠堂脊最后）；祠堂北端那片黑随真祠堂沉（机器唯一没盖住之处），未补数据仍存硬盘、命名留空。数字南屿交付合格（陆："形留住了"）。江照带名册与"钟满"之名，搭周晏的铁皮船，从南屿划到老浔口——**替祖父划完那趟"上不去"的返程**：河心把号牌"浔渡贰号"贴水而过，把名字带到"那头"（老浔口对岸半坡），抱名册上岸。
- **ch12 签收（蓄水后，10/11交付）**：数字南屿归档（何科员"收讫"）。模型里西三巷十一号是补面空墙，手一伸就穿过去（echo ch02）。江照成守册人，按钟阿婆法子手抄名册；去库区把老浔口渡口桩顶坐标放样（编号1976，离岸41米、水下），据此给钟满"葬处"栏补上一组一比一的坐标（X3104627.318/Y35486213.905/H251.476）——测绘头一回用于"记人"。扉页"签收人"签下江照+二〇二九年十一月。设备清单号牌"用途"栏（空了一路），终于填一个字：**渡**。钟阿婆在邻县侄女家（信/物/一次探望，深化钟满"过河抓药、让船"的原委），赠江照原来包名册的蓝布。

## Exit state（全书出口）
南屿沉、老浔口二次没顶。数字南屿交付合格却空心。江照：守册人，替钟满补了"葬处"（水下坐标）、在名册签了"签收人"、给号牌"用途"填了"渡"。钟阿婆异地安居。账未平（命锁死53年），但被人接住、落了地。

## Character deltas
- 江照：由"信点云＝留住"到承认留不住；以"人来记"接下机器接不住的（签收名册、带名过水、坐标记人）。测绘从"记地方"到"记人"。
- 钟阿婆：把守了一辈子的名册交出、离谷而非殉谷；受住了那笔账（不赦不罪）。
- 周晏：始终"接受/了断"，撑船助渡，"到了"如"了了"。

## New world rules（本卷）
- 二次淹没机制坐实：老浔口先被老坝淹（1976），南岭再压约40米。
- 数字南屿以"业主未授权＝拒扫不纳入"合法地留白（钟阿婆屋）；补面可抹平梁上的黑而过验收。
