# Volume Digest — 《一比一》 vol 1: 测绘

<!-- HARD CAP 2,500字。卷二起草加载此摘要，替代卷一原文/账本。 -->

## What this volume did for the book（一句话）
把江照的"记录＝留住"信仰立起来，又让它撞上名册、无坟者与自家旧账，逼她从"扫描留形"迈向"用手记名"的第一步。

## Plot summary（按章簇）
- **ch01–02（到站·点云，D97→D95）**：2029年，测绘员江照到浔江河谷南屿村，接手"数字南屿"1:1 扫描项目（100天后蓄水淹没，水深约40米；晒场高程286，淹没线326）。本地联络周晏在清自家坟头、备迁三代祖坟。江照贴身带祖父江守遗物——渡船铜号牌"浔渡贰号"（背编号磨平），未言明来历。城里对接人姓陆，验收定在蓄水前十天、约60天后整村交、数据为唯一存档。周晏手"穿过"点云房梁——点云是空壳，记形不记里。守碑老人钟阿婆（钟秀英，78，西三巷十一号）拒扫，问"照得住名字么"。祠堂后进正梁北端有一片六站中四站激光未回、补不上的黑，朝后山坟地方向（未解，江照未补）。
- **ch03（名册，D88）**：钟阿婆出示手抄《浔口名册》（一本，牛皮纸包皮、毛边内页、她手书封面），逐名记生卒葬处。中段二三十行"殁年一九七六、葬处空"＝老浔口被上一座坝放水淹死、至今无坟者；五十来年只补上三个（多为经她验看过的衣冠冢）。钟阿婆立"签收"母题："东西送到得有人签收"；并按空条轻言"这上头也押着我自个儿的人"（未指名）。江照在空条里瞥见一个"江"姓（同己姓），按成"巧合"，未碰号牌。周晏旁证老浔口在下游十几里、二次淹没同一地方。
- **ch04（迁坟，D71）**：后山坟地（坡高程313–330，线326）线下147座须限期迁，逾期由工程队机械起、装编号箱入县公墓；线上二十几座不管。周晏人工迁自家三代祖坟（"了了"）＝了断/镜子。江照以点云断口比对为黄家断碑正名定位（记录之"用"，六年头一回觉值当）。钟阿婆持名册拦机械起何广仁衣冠冢——其冢系堂弟自作、未叫钟阿婆验看，故名册葬处仍空；起出半副泡糠水骨"对得上名字对不上人"，随葬豁口搪瓷缸。江照记录本"下落"栏填不出、留空。
- **ch05（号牌，D58夜）**：三叔来电办祖父江守身后事（销户、寄存单末三位307、分遗物）。号牌来历落实：江守是老浔口那夜摆渡人；临终糊涂半句"贰号还系在桩上，我一趟就上不去了"。江家那年离谷、抹去籍贯、绝口不提。**江照首次对自己承认**：主动按下南屿一栏是为私账、为在二次没顶前看一眼江家的谷（老浔口＝江家的谷）。设备清单号牌"用途"栏仍空。
- **ch06（水位，D43，卷climax）**：江照借看名册比日子。农历丙辰年（1976）六月十九（公历八月十四）那一夜同日压14名溺者空条；其中**钟满，13岁**，钟阿婆认"我兄弟"。那夜桥断、渡船"来了一趟、载一船过去就没再回来"，钟满在这头没上去。江照以"两独立量落同点＝互相印证"确认：祖父"只运一趟就上不去"那夜＝钟满溺死那夜。**当场未交底**（号牌留内袋，钟阿婆未认出、不知江照身世）。江照**取消轮岗、留到蓄水**（对外报"逐日记录水位/最后所在"，属实但非真因）。章末：弃仪器键盘，用签收单那支粗笔在记录本背面硬纸上一笔一画手写"钟满"，压号牌，合本。

## Exit state（出口状态，D43）
江照知己身来历（那个没回头摆渡人的孙女），留在南屿等蓄水；已手写下第一个名字。钟阿婆守名册与碑、不肯离谷，认了江照来比日子，但不知其身世、不知那块号牌。周晏已迁完自家坟，在下游清最后无主坟。外业扫描基本收尾（除钟阿婆屋/人未扫）。

## Character deltas
- 江照：信"点云＝留住" → 认出身世、放弃轮岗留下、首次以"手写记名"对抗"机器留形"。代价：测绘不再是中立的活。
- 钟阿婆：独守名册、有个人失亲之痛（未明说）。
- 周晏：了断自家三代坟，"接受"以衬江照的"留存"。

## New world rules（本卷新立）
- 点云空心（手可穿过、记形不记里/去向）；"补面"以旁料填洞过验收。
- 名册规则：衣冠冢须经钟阿婆验看方补葬处，未验则葬处空。
- 二次淹没：老浔口1976被上一座坝淹一次、将被南岭淹第二次（同址再深40米）。
