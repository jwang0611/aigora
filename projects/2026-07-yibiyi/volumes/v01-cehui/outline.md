# Volume Outline — 《一比一》 vol 1: 测绘

**Cap**: 5,000字
**Locked at**: V.OUTLINE checkpoint (D-006, decisions.md). In-volume beat 偏移允许，除非改出口状态。
**depends-on**: —（首卷，无前卷 digest）

## Volume Arc（卷弧）
- **Volume question**: 一座村子能不能靠"被复刻"而真正留住？江照的测绘信仰能不能兑现？
- **Entry state**: 2029年，南屿蓄水倒计时约100天起。江照到站，"数字南屿"1:1 扫描项目启动。
  她贴身带着祖父江守的铜号牌"浔渡贰号"，不肯承认为何指定接这活。
- **Mid-volume turn**: 江照撞见钟阿婆手抄的《浔口名册》与上一座坝（老浔口）的无坟溺者；
  点云的"空心"与名册的"实在"对峙。
- **Volume climax → exit state**: 江照借号牌与名册，认出无坟男孩钟满、以及祖父那一夜——家族旧账落地。
  永久改变：她放弃轮岗撤离，决定留到沉没。倒计时约剩40天。这是全书"从记录到承担"的第一步。
- **Volume-end hook**: 她知道了自己是谁；钟阿婆尚不全知。她留下来，为的不是扫描。
- **Promises due this volume**: PROM-002 pay（v1ch06 认出）；PROM-001/003/004/005/006 plant/escalate。

## Beats（逐 beat，一章一 beat）

### ch001 到站
- **Situation → change**: 江照坐末班中巴到南屿，接手 1:1 扫描项目；见工程队在墙上刷红漆淹没线、
  在向导图上把钟阿婆的碑标为"障碍物"。她对着倒计时牌与比例尺，立下"1:1 完整留住"的职业信念。
  章末：她远远看见一个老太婆蹲在一块红漆碑前，用毛笔往一个本子上添字。
- **Reader learns/feels**: 100天倒计时；1:1 承诺；主角的测量信仰；她贴身有个说不清来历的铜号牌。
- **Ledger-critical facts**: 倒计时=100天起（本章设为"还有97天"）；淹没线红漆；号牌"浔渡贰号"贴身带；
  项目名"数字南屿"；江照职业=数字化测绘员。
- **cast**: 江照, 周晏（接站/本地联络）, 钟阿婆（远景）
- **promises**: PROM-001 plant / PROM-002 plant（号牌·指定接活）/ PROM-004 plant（本子=名册远景）/ PROM-005 plant（老太婆留守）

### ch002 点云
- **Situation → change**: 扫描作业。江照给周晏演示平板上的"数字南屿"——旋转、放大、1:1。周晏把手
  "穿过"屏幕里的房梁：点云是空心的。江照去扫钟阿婆的老屋，被拒之门外；老太婆只说了句关于"名字"的话。
  章末：江照在点云里发现祠堂梁上一处扫描补不上的暗影（后山坟地方向），存疑。
- **Reader learns/feels**: 点云"像而无内、手可穿过"；副本记录不了"里面/去向"；钟阿婆拒扫、护着一个本子。
- **Ledger-critical facts**: 点云空心（手穿过）；钟阿婆拒绝被扫描；江照住工程队板房。
- **cast**: 江照, 周晏, 钟阿婆
- **promises**: PROM-001 escalate（空心初显）/ PROM-004 escalate（名册·名字）

### ch003 名册
- **Situation → change**: 江照为扫后山坟地找钟阿婆带路。老太婆拿出《浔口名册》，一笔一画的死者账，
  含一批"只有名字、没有葬处"的空条——1970s老浔口放水淹死、无坟的人。江照被一个姓氏刺到（与己同源），
  嘴上说是巧合。周晏旁证：老浔口，上一座坝，二次淹没。
- **Reader learns/feels**: 名册=唯一记"名字与下落"的册子；无坟溺者的存在；江照对某姓氏的异常反应。
- **Ledger-critical facts**: 《浔口名册》有无坟空条；老浔口1970s被淹；"二次淹没"设定确立；
  江照见到一个让她失态的姓氏（先不点名钟满，埋"钟"姓/或与江家相关的一笔）。
- **cast**: 江照, 钟阿婆, 周晏
- **promises**: PROM-003 plant（无坟者·空条）/ PROM-002 escalate（姓氏刺激）/ PROM-004 escalate

### ch004 迁坟
- **Situation → change**: 迁坟工地。淹没线下的坟须限期迁走，逾期由工程队"统一处理"。周晏迁自家三代祖坟，
  平静利落。一户人家的坟无人认领，工程队要机械起走；钟阿婆来拦，因为册上有名。江照第一次用她的"记录"
  帮上忙（比对名册与坐标，替一座坟正名/定位），也第一次撞见记录之外的东西——骨殖、旧物、说不清的下落。
  章末：一具无名骨殖，名册上对得上名字、对不上人。
- **Reader learns/feels**: 迁坟的制度冷酷与倒计时的压迫；江照的测绘能力真有用（不是愚蠢）；
  但"对得上名字、对不上人"的裂缝。
- **Ledger-critical facts**: 逾期坟由工程队统一处理；江照用扫描/坐标协助定位坟；出现一具名册对得上名字
  但下落存疑的骨殖。
- **cast**: 江照, 周晏, 钟阿婆
- **promises**: PROM-003 escalate / PROM-001 escalate（记录的用与限）

### ch005 号牌
- **Situation → change**: 夜。江照接到城里电话（处理祖父江守遗物/尾事），带出号牌来历：江守曾是老浔口
  摆渡人，那一夜之后举家离谷、绝口不提。江照终于对自己承认：她指定接南屿，是为了这桩没头没尾的旧账。
  她把号牌拿出来看——"浔渡贰号"。她意识到老浔口就是江家的谷。
- **Reader learns/feels**: 身世逼近落地——祖父=那夜摆渡人；江照来此的真正动机；号牌的分量。
- **Ledger-critical facts**: 祖父江守=老浔口摆渡人；号牌"浔渡贰号"；江家那夜后离谷、不提；
  江照动机=私账（首次向自己承认）。
- **cast**: 江照（独场为主）, 周晏（少量）
- **promises**: PROM-006 plant（号牌·未完成的一渡）/ PROM-002 escalate

### ch006 水位（卷climax）
- **Situation → change**: 江照带号牌去见钟阿婆（先不全交底），旁敲名册里那批无坟空条的日期。她认出：
  其中一条是**钟满，13岁**，溺死之日正是祖父摆渡的那一夜——祖父运了一趟就再上不去，钟满没过来。
  账落地：她祖父落下的那个孩子，就在钟阿婆的册子里、就是钟阿婆的弟弟。江照当场没有交底（承受不住），
  但她做了一个决定：取消轮岗，留到蓄水那天。章末动作：她在自己的测绘记录本背面，第一次手写下三个字——
  一个名字（钟满），像名册那样，一笔一画。（对位：从点云到手写。）
- **Reader learns/feels**: PROM-002 pay——江照与谷子/钟阿婆的关系明；钟满之死与祖父那夜的因果；
  她选择留下、且开始"用手写记名字"。
- **Ledger-critical facts**: 钟满=钟阿婆之弟，13岁，那夜溺死无坟；祖父那夜"只运一趟"；江照取消轮岗、
  留到沉没；江照首次手写名字（记录方式转变）。倒计时约40天。
- **cast**: 江照, 钟阿婆
- **promises**: PROM-002 pay / PROM-003 escalate / PROM-006 escalate / PROM-005 hold
